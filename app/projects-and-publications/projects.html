BrailleToTextTranslator

Translate Braille Writing to Plain Text

The tool takes in images of Braille writing, applies image processing and pattern recognition, and converts it to simple text. As the braille dots are simply embossed on paper, a scan of it produces a small gradient difference between the background and the dots. So, we tuned a pre-processing phase with image enhancement and noise reduction technics for making the dots differentiable and clear. The script could have a rotation that might have occurred during the writing and/or the scanning. For this, we proposed a rotation-invariant geometric structure recognition method which could also help in reconsidering braille dots that were incorrectly added or removed during the pre-processing phase.

ResultAnalysisTool

A Complete Package for Processing OMR Answer Sheet

The software takes in a batch of scanned images of OMR sheets and extracts students’ answers into a CSV file. The user then can mark different fields of the extracted data, such as class, roll, answer, and so on. Lastly, the score is calculated by comparing the students’ answers with real answers.

Egami

A Light Weight Image Processing Library

This library implements various base image processing algorithms, ranging from image enhancement and filtering to feature extraction. It is specially suitable for small projects. We used this library in several of our own projects such as BrailleToTextTranslator, ResultAnalysisTool, GenderRecognizer, and so on.
